# HR Analytics Dashboard

## Project Overview
This project presents an HR Analytics Dashboard built using Microsoft Excel. It analyzes employee data to provide insights into workforce distribution and salary trends using Pivot Tables and Charts.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Google Sheets

## Dashboard Features
- Total Employees
- Average Salary
- Employee Count by Department
- Average Salary by Department
- Interactive Dashboard Layout

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Pivot Tables
- Data Visualization
- Dashboard Design
- HR Analytics
- Excel Reporting
